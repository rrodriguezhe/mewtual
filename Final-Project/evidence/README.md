# Evidence

Supporting evidence that each implemented user story (from `docs/analisis.pdf`, section 4)
actually works, as required by the brief's "Implemented User Stories with Supporting Evidence."

One screenshot (or short clip) per delivered HU is enough — showing the acceptance criteria
being met, not just the screen existing. Name files by story so they're easy to cross-reference
from `final-report.pdf`, e.g.:

```
HU-01-registro.png
HU-02-login.png
HU-06-cruce-responsable.png
HU-07-08-swipe.png
```

Only include stories that are actually implemented and testable — don't fabricate evidence for
stories still in progress.
